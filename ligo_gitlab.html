Generating and using an SSH key

  • Click on the icon in the very top right (with a small dropdown arrow next to it) -> settings -> SSH keys
  • Add an SSH key or follow the instructions on how to generate one and then add it to this website. The instructions are here

Fork and Pull

  • Generally we use the fork/pull method
    • Click "fork" on the repository you want to use
    • This will create a version on your own account
    • Clone your version, make edits and commits to that version
    • Then submit a pull request with my version. I will review it and either make some changes to it, ask you to make changes, or accept it.
